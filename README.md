# TTA Leaderboard

Static site bundling one result snapshot as `dist/data.json`, rendered by
`dist/index.html` (Bootstrap + vanilla JS, no build step).

## Rebuild the data

Scans `results/**/*.json` and re-emits `dist/data.json` (re-run after every
`scripts/sync_results.sh` / `scripts/sync_and_build.sh`):

```bash
python3 leaderboard/build_static.py [results_dir] [out_path]
# defaults: results_dir=results  out_path=leaderboard/dist/data.json
```

`papers.json` (the 19 benchmarked methods' BibTeX) is an input to `build_static.py`
and feeds the Details tab's LaTeX `\cite{}` / BibTeX exporters. Required.

## Preview locally

Serve `dist/` on `http://0.0.0.0:7860` (no-cache headers):

```bash
python3 leaderboard/serve.py
```

## Layout

```
leaderboard/
├── build_static.py     # results/ → dist/data.json  (called by scripts/)
├── serve.py            # static file server for dist/
├── papers.json         # benchmarked-method BibTeX (Details-tab export input)
└── dist/               # deployed static site
    ├── index.html      # the site (self-contained, reads data.json)
    ├── bootstrap.min.css
    └── data.json       # generated by build_static.py
```
